## 기능 ##
- 아이템에 따라 점수 획득, 체력 감소 처리
- HELLO WORLD 젤리를 모두 수집하면 젤리 천국으로 이동
- 장애물을 점프해서 피하거나 선생님을 마주치면 인사를 하여 진행
